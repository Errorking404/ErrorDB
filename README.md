# ErrorDB

**ErrorDB** ek tool hai jo sim owner ki information jaise:

- **Name**
- **CNIC**
- **Address**
- **Sim ki Company**

ki details deta hai. Ye tool **free** hai.

## Installation

### Copy and Paste Commands

```
git clone https://github.com/Errorking404/ErrorDB.git
cd ErrorDB
python install.py
```
**Run:**
```
python Error.py
```
